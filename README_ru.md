# Космическая сингулярность ✨

> Постепенная игра в жанре idle game, в которой вы извлекаете темную материю из квантовой сингулярности, возноситесь и преодолеваете Большой взрыв.

Этот проект представляет собой уникальный взгляд на жанр «idle game», фокусируясь на космической тематике и сложной игровой механике. Он создан с использованием современных веб-технологий, предлагая визуально привлекательный и глубоко стратегический игровой процесс.

## О 🌌

Cosmic Singularity — это инкрементальная игра в жанре idle, в которой игроки управляют извлечением тёмной материи из квантовой сингулярности. Игра включает в себя стратегические улучшения, механику престижа и уникальную систему развития, охватывающую вселенские циклы. Она призвана обеспечить глубокий и увлекательный опыт для игроков, которым нравятся сложные симуляционные игры и игры с управлением ресурсами.

## Особенности 🚀

- **Глубокая инкрементальная механика:** продвижение по уровням улучшений и системам престижа.
- **Взаимодействие с квантовой сингулярностью:** Щелкните и управляйте центральной сингулярностью для генерации ресурсов.
- **Система вознесения:** сбросьте прогресс, чтобы получить постоянные бонусы (осколки) и разблокировать новые уровни способностей.
- **Улучшения престижа:** Вкладывайте осколки в дерево навыков, чтобы улучшить основные игровые механики, такие как скорость кликов и эффективность кликов.
- **Система достижений:** Разблокируйте различные достижения по мере прохождения разных этапов.
- **Сохранение и резервное копирование:** управляйте несколькими файлами сохранений и создавайте резервные копии для обеспечения безопасности.
- **Прогресс в автономном режиме:** игра рассчитывает и применяет прогресс, достигнутый игроком во время его отсутствия.
- **Визуально насыщенный интерфейс:** используется космическая, неоновая эстетика с плавной анимацией.

## Технологический стек 🛠️

- **Язык:** TypeScript
- **Фреймворк:** React, Next.js (TanStack Router)
- **Стилизация:** Tailwind CSS
- **Управление состоянием:** React Hooks, React Query
- **Инструмент сборки:** Vite
- **Компоненты пользовательского интерфейса:** Radix UI, Shadcn/ui, Vaul, Sonner, Lucid React.
- **Анимация:** Framer Motion
- **Проверка синтаксиса и форматирование:** ESLint, Prettier
- **Сервер:** Node.js (через Nitro для SSR)

## Монтаж 💻

Этот проект представляет собой веб-приложение и может быть запущен с использованием Node.js и npm/yarn/pnpm.

1. **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/Nathan-Pro-FR/cookie_clicker.git
    cd cookie_clicker
    ```

2. **Установите зависимости:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3. **Запустите сервер разработки:**

    ```bash
    npm run dev
    ```

4. **Откройте в браузере адрес** `http://localhost:5173` (или порт, указанный Vite).

## Использование 🎮

Cosmic Singularity — это игра в жанре «idle game», в которой вам предстоит кликать мышкой и управлять ресурсами для достижения космического господства.

1. **Нажмите на сингулярность:** ваше основное взаимодействие заключается в нажатии на центральную сингулярность для генерации темной материи.
2. **Покупка модулей:** Используйте накопленную темную материю для покупки модулей, которые пассивно генерируют больше темной материи с течением времени (CPS - космическое производство в секунду).
3. **Престиж (Вознесение):** Достигнув определенного порога общего количества Темной Материи, вы можете инициировать «Большой взрыв» (вознесение), чтобы сбросить текущий прогресс, но получить Космические Осколки.
4. **Тратьте осколки:** используйте Космические осколки для покупки улучшений престижа, которые обеспечивают постоянные бонусы к вашему CPS, силе кликов и скорости получения осколков.
5. **Разблокируйте достижения:** Стремитесь выполнить различные достижения, которые отслеживают ваш прогресс и этапы.
6. **Сохранение и загрузка:** Используйте слоты сохранения и функции резервного копирования для управления своим игровым прогрессом.

### Пример игрового цикла 🔄

- Для начала нажмите на сингулярность, чтобы накопить начальное количество темной материи.
- Для начала пассивной генерации приобретите первые несколько модулей "Sonde Orbitale".
- По мере увеличения вашего CPS покупайте более продвинутые модули, такие как «Moissonneur de Naines Blanches» и «Manipulateur de Cordes Cosmiques».
- Чтобы разблокировать вознесение, стремитесь к достижению "Стабильность сингулярности" (1 миллион единиц темной материи).
- Поднимитесь наверх, чтобы получить Осколки, а затем вложите их в улучшения, такие как "Vitesse de la Lumière" (+25% CPS) и "Noyau Obscur" (удвоенная сила кликов).
- Продолжайте играть, развиваться и улучшаться, чтобы достичь новых космических вех.
